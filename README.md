<h1 style="color:red;">Block Shooter AR Game</h1>

<h2 style="color:red;">Introduction</h2>
Block Shooter AR is an interactive augmented reality game where players can build structures using virtual blocks and then destroy them using a shooting mechanism. The game combines creativity and action, allowing users to design freely and experience realistic physics-based destruction in an AR environment.

<h2 style="color:red;">Objectives</h2>

<h3 style="color:red;">Primary Goal</h3>
- Create an engaging AR experience that blends building and destruction mechanics.

<h3 style="color:red;">Learning Objectives</h3>
- Understand and implement augmented reality concepts  <br> 
- Develop interactive object placement systems <br>
- Apply physics for realistic object behavior and destruction <br>
- Design intuitive user controls for AR environments <br>

<h2 style="color:red;">Game Description</h2>

<h3 style="color:red;">Gameplay</h3>
- Players place virtual blocks in the real-world environment using AR   <br>
- Structures can be built freely with the provided blocks <br>
- Once the structure is complete, players can press the shoot button   <br>
- Projectiles are launched to knock down and destroy the structure <br>
- Realistic physics simulate collapse and impact   <br>

<h3 style="color:red;">Features</h3>
- AR-Based Building System – Place blocks in real-world space <br>
- Freeform Construction – Build any structure creatively   <br>
- Shooting Mechanism – Fire projectiles to destroy structures   <br>
- Physics Simulation – Realistic collisions and destruction <br>
- Interactive Controls – Easy placement and shooting system <br>

<h2 style="color:red;">Development Process</h2>

<h3 style="color:red;">Tools Used</h3>
- Engine: Unity   <br>
- Language: C#   <br>
- AR Framework: AR Foundation / ARCore / ARKit   <br>
- Assets:   <br>
  - 3D block models <br>
  - Particle effects   <br>
  - Sound effects   <br>

<h3 style="color:red;">Key Components</h3>

<b>AR System</b>  
- Plane detection for placing objects in real-world space  

<b>Blocks</b>  
- Placeable objects with colliders and rigidbody components  

<b>Shooting System</b>  
- Projectile launcher triggered by a shoot button  

<b>Physics Engine</b>  
- Handles collisions, gravity, and object destruction  

<b>Scripts</b>  
- PlacementController.cs – Handles AR block placement <br>
- ShootController.cs – Manages shooting mechanics   <br>
- PhysicsManager.cs – Controls physics interactions <br>
- GameManager.cs – Oversees game flow   <br>

<h3 style="color:red;">Workflow</h3>
1. Set up AR environment with plane detection   <br>
2. Implement block placement system <br>
3. Add shooting mechanics with projectiles   <br>
4. Apply physics for realistic destruction <br>
5. Enhance with sound and visual effects   <br>

<h2 style="color:red;">Core Concepts</h2>

<b>Augmented Reality</b>  
- Virtual objects are placed in real-world environments using device camera  

<b>Object Placement</b>  
- Blocks are positioned using detected surfaces  

<b>Physics Simulation</b>  
- Gravity and collisions simulate realistic destruction  

<b>User Interaction</b>  
- Touch input for placing blocks and triggering shooting  

<h2 style="color:red;">Challenges Faced</h2>
- Accurate AR plane detection and object placement   <br>
- Maintaining performance with multiple physics objects   <br>
- Ensuring stable and realistic destruction behavior   <br>

<h2 style="color:red;">Results</h2>
- Developed a functional AR game with building and shooting mechanics   <br>
- Successfully implemented interactive block placement <br>
- Achieved realistic physics-based destruction   <br>
